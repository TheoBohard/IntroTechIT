Les différentes faces du web

Le internet est un ensemble de reseau interconnectés grâce à différente technologie. c'est un reseau de reseaux.

Le web ou la toile est une application d'internet comme les mail, les messagerie instantanées ou encore le partage Pea-to-Pear.

Cette toile est sépraré en plusieurs "partie" celon la nessessité d'accessibilité, les technologies ou encore la volonté du détenteur du site ou réseau.

Le web surfacique

Le web dit surfacique ou visible est le web accessible via un moteur de recherche standard

Le Deep web

Le web profond, appelé aussi toile profonde ou web invisible décrit dans l'architecture du web la partie de la toile non indexée par les principaux moteurs de recherche généralistes.

Le Dark Web

Le dark web, aussi appelé web clandestin ou encore web caché, est le contenu de réseaux superposés qui utilisent l'internet public, mais qui sont seulement accessibles via des logiciels, des configurations ou des protocoles spécifiques.
