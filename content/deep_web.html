Deep web

Le Deep web ou web profond correspond à 90, 95% du web.

Il contient toutes les pages web et informations non indexés par les moteurs de recherche aussi bien ce qui ne veut pas etre montré que ce qui ne peut pas être montré.

NON LIÉ
Certaines pages webs n'ont pas de lien avec d’autre pages. Les robots d'indexations ne peuvent donc pas les parcourir et les ajouter au catalogue des moteurs de recherches.
Script
Certaines pages webs contiennent des scripts javascripts qui bloquent involontairement l'accès aux robots d'indexations.
Non indexable
Certains formats sont incompréhensibles par les moteurs de recherche. Avant 2008, il n'etait par exemple pas possible d'obtenir un pdf en resultat de recherche. Les format reconnus au départ étaient le format natif du web : le html.
Trop volumineux
Certains sites acceuillants de grandes bases de données ne sont pas reconnus par les robots d'indexations a cause de leurs contenus trop volumineux. Le National climatic data center (370To de données) n'est par exemple pas completement reférencé. A partir de 500ko de données, les index de sites ont plus de mal a être référencé correctement
Privé
Il est possible de bloquer l’indexation avec le fichier contenant les droits à l'indexation “robots.txt”.
Accès limité
Certains sites web nessessitent une action "humaine" pour pouvoir avoir accès au contenu :
  • Authentification et identification (login et mot de passe)
  • Captcha
  • Formulaire
IOT ( Internet des objets )
Les objets connectés ne sont pas indexés par les moteurs de recherches même si ils possèdent une URL. Cela permet d'eviter d'indexer l'inutile. Cette information est généralisé à tous les moteurs de recherches classiques comme google, on retrouve cependant aujourd'hui des moteurs de recherches pour les objets connectés comme sodan.
Dynamique
Les sites webs contiennent de plus en plus de pages dynamiques. Par exemple les formulaires de réservations de billets trains s'adaptent en fonction de la date demandée. Si la date du premier Mars 2022 est beaucoup recherchée, elle ne sera indexé pour autant. C'est la page mère qui le sera.
Nom de Domaine non standardisé
Les sites webs dont la resolution DNS n'est pas enregistrée chez l'ICANN ( Internet Corporation for Assigned Names and Numbers ) nes sont pas indexés par les moteurs de recherche.
